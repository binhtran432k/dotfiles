# Installs OpenSUSE

## Build Tools

```sh
zypper install make gcc tar
```

## Utilities

```sh
zypper install curl eza lazygit yazi fd fzf ripgrep bat git-delta
```

## Editor

```sh
zypper install neovim tree-sitter
```

## Shell

```sh
zypper install fish starship
```

## Dev

### NPM + Node

```sh
zypper install fnm
```

### Zig + Zls

```sh
zypper install zvm
```

### Rust + Cargo

```sh
zypper install rustup
```
